## API Integration

Used `createAPI` feature from Redux toolkit to make API requests.

## Set Up

Create a `.env` file and populate with `VITE_API_URL=https://fakestoreapi.com`

In the project directory, you can run:

#### Installation

`npm install`

#### Run App

`npm run dev`

#### Build App

`npm run build`
